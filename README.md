## Setup

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd hackathon-book-ai
   ```

2. Navigate to the website directory:
   ```bash
   cd website
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run start
   ```
